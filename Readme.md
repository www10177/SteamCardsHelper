## Introduction
卡片的數量是抓取Steam市集上和該遊戲相關有上架的卡片，若有卡片都沒人上架的話，可能會導致推算的結果(所需寶珠數、成本)錯誤  
卡片的售價抓取Steam 市集內該卡賣方掛單的最低售價  
寶珠的價格亦是取賣方掛單的最低售價
